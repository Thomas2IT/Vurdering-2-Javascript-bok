Break

Break sørger for at kontrolløkken den tillhører stopper uansett hva som skjer inne i den.

For å vise hvordan det funker har jeg følgende eksempel:





Koden er utbredt, og navnene er byttet om og oversatt, men koden er opprinelig hentet fra https://www.w3schools.com/jsref/jsref_break.asp


Trykk på knappen for å kjøre eksempelet. Koden ville vanligvis printet ut 1-5, men siden vi har satt en break i den vil den slutte når variabelen sin verdi er det samme som 4.






    function break_eksempel() {
      var text = "";
      var antall = 0;
      while (antall < 5) {
        text += "
Nummeret er " + antall;
        antall++;
        if (antall === 0) {
          break;
        }
      }
      document.getElementById("demo").innerHTML = text;
    }

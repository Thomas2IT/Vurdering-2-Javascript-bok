Continue

Continue er brukt i de tilfellene der du vil hoppe over noe i løkken. Om vi for eksempel har en løkke som teller fra 1 til 4, så vil man kunne bruke continue for å hoppe over et tall i rekken mellom 1 og 4.


Koden er utberedet, og navnene er byttet om og oversatt, men koden er opprinelig hentet fra https://www.w3schools.com/jsref/jsref_continue.asp

Trykk på knappen for å kjøre en loop som teller til 4, men som hopper over 3 på grunn av continue.

    function C0ntinue() {
        var text = "";
        var antall;
        for (antall = 0; antall < 5; antall++) {
          if (antall === 3) {
            continue;
          }
          text += "Nummer " + antall + "
";
        }
        document.getElementById("demo").innerHTML = text;
      }
